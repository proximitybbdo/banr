Banr

Banner alternatives made easy

Download at Github

About

So you make Flash banners, and then you have to create the alternative jpg? It's always a hassle when you have to a whole batch of them, right? Wrong! Use Banr to do it for you. Just drag in your swf files and wait for the jpg's to be spit out.

Installation

You need the Adobe Air runtime to run the app, which you can grab from Adobe for free. Then just download the latest version at the top here.

More

If you want more or different features, you can make a fork request to help make Banr better, or just make your own. All is on github.
